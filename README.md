# 🃏 Memory Game

A fun card-matching game where players match pairs before time runs out.

## 🚀 Features
- Card matching gameplay  
- Timer countdown  
- Score tracking  
- Reset game option  
- Interactive UI  

## 🛠️ Tech Used
- HTML  
- CSS  
- JavaScript  

## ▶️ Run
Open `index.html` in a browser.

## 📂 Files
- index.html  
- style.css  
- script.js
